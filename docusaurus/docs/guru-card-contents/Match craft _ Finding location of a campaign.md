# Match craft   Finding location of a campaign

Step 1: After logging in to Matchcraft through this link 
, 
https://digitaladvertising.advertiserreports.com/
we need to search for the account in the marked section shown on the screenshot.

![](/img/guru/Match%20craft%20_%20Finding%20location%20of%20a%20campaign/01.png)

Step 2: Once the account appears, click on the active account and the below page will open up. The 
marked section contains all the active campaigns under the account. Select a campaign from that 
section.

![](/img/guru/Match%20craft%20_%20Finding%20location%20of%20a%20campaign/02.png)

Step 3: Once we select any of the campaigns, the section shown below will open up. This is where 
we can see the location list. To elaborate more, we can click on "Specify by radius."

![](/img/guru/Match%20craft%20_%20Finding%20location%20of%20a%20campaign/03.png)

Step 4: This is how it will appear in this section.

![](/img/guru/Match%20craft%20_%20Finding%20location%20of%20a%20campaign/04.png)
