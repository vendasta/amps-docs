# Website Content Creation Form

## How to find the Website Content & Creation Form using Work Order System

Search in Zendesk using the name of the account or AGID and look for the 'work order created' ticket.

![](/img/guru/Website-Content-Creation-Form/01.png)

If there are many to choose from similar to this, you can hover over the ticket and look for the main
order i.e. (Templated Website)

![](/img/guru/Website-Content-Creation-Form/02.png)

Open the ticket and click View work order.

![](/img/guru/Website-Content-Creation-Form/03.png)

On this screens click fulfillment form.

![](/img/guru/Website-Content-Creation-Form/04.png)

This next page will contain all the information including the C&C Form

![](/img/guru/Website-Content-Creation-Form/05.png)
