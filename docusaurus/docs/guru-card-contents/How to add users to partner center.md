# How to add users to partner center

Go to the account you want in partner center

Find this section in the account page called users

Click add users

![](/img/guru/How%20to%20add%20users%20to%20partner%20center/01.png)

Click create user

![](/img/guru/How%20to%20add%20users%20to%20partner%20center/02.png)

Fill in the information!

![](/img/guru/How%20to%20add%20users%20to%20partner%20center/03.png)

  Click save and the user will be added!
