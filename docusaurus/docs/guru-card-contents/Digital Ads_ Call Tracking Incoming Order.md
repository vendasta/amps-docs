# Digital Ads Call Tracking Incoming Order

When we receive this order, we don't see any fulfillment status. We have to create it manually.

Please navigate to VTM and click on the three dots as indicated below.

![](/img/guru/Digital%20Ads_%20Call%20Tracking%20Incoming%20Order/01.png)

Next, we need to add 'Digital Ads: Call Tracking' in the Name column as depicted below and assign it 
to the Digital Ads specialist.

![](/img/guru/Digital%20Ads_%20Call%20Tracking%20Incoming%20Order/02.png)

Additionally, include the task 'Send Confirmation Email' and assign it to the individual responsible for 
Admin tasks.

![](/img/guru/Digital%20Ads_%20Call%20Tracking%20Incoming%20Order/03.png)

Click on Create Project.

![](/img/guru/Digital%20Ads_%20Call%20Tracking%20Incoming%20Order/04.png)

Create two Zendesk tickets: one for sending an order confirmation email to the PID assignee, and the 
other to assign a specialist thread to the DA specialist.
