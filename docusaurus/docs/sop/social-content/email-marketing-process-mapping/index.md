# Email Marketing Process Mapping

Use the tables below as quick-reference checklists. Narrative detail from the source SOP is preserved in the **Details** column where it helps on screen.

---

## Email Marketing Process

### Standard — journey at a glance

| Stage | What happens | Notes |
| --- | --- | --- |
| **Order** | Put the order on the lifeline | Within **24 hours** |
| **Onboard** | Booked → finished → receive onboard from comms | Sequential |
| **Monthly project** | Stand up the monthly project | Can be done by the **system**, **us**, or **comms** (not determined yet) |
| **Initial strategy** | Deliver initial strategy | **5 days** (non-approval) |
| **Comms** | Sends initial strategy & content together? | Confirm with comms |

### Standard — proposal creation

| Step | What to do |
| --- | --- |
| 1 | Open the template and **make a copy**. |
| 2 | Change branding to match the business’s **colors** using their website as reference. |
| 3 | Use **ChatGPT** for target audience, social platforms, content pillars, SMART goals, and **~3 months** of content ideas. |
| 4 | Download as **PDF** and send to **Client Engagement** for the onboard call (SMM+ upgrade story). |

### Standard — Campaigns Pro: get to the email builder

| Step | Action |
| --- | --- |
| 1 | Go to the client’s **business app** and **Campaigns Pro**. |
| 2 | **Partner center** → partner **PID**. |
| 3 | **Accounts** → search the **SMB** → **Manage Accounts** → type SMB name. |
| 4 | Top of screen **Open in** → **Business App** (wording may differ if white-labeled). |
| 5 | In **Business app** → **Campaigns**. |
| 6 | In **Campaign Pro** → **Create Campaign** → name it. |
| 7 | **Add Email** → **Add New Email**. |
| 8 | **Template name** = series name (e.g. “Welcomes Series”). |
| 9 | Add **subject line** and **inbox text** (order flexible). |
| 10 | **Add New Block** for content; match **brand** colors. |
| 11 | **Save** → finish remaining emails → **automations** → **timezone** → **send schedule**. |

*Confirm the exact scheduling / automation wording with your team if your source template differs.*

### Standard — sending strategy for approval

| Do this | Details |
| --- | --- |
| Find cohort | Check the **lifeline** for your account’s cohort. |
| Verticals | **Agency** · **High Velocity** (use the right lane). |
| Recipients | Address to the correct cohort and **cc** as required. |
| Attach | Strategy proposal + **content calendar** + **content link**. |
| Subject line | `ATTN: [Comms person name] [PID]: [Business Name] - Strategy and Content Proposal for Approval` |
| Body copy | `[PID] [Business Name]`’s strategy proposal and first email campaign are ready to review — feedback/edits due by **[due date]**. |
| Gate | **Wait for approval** before drafting email content. |

## Official strategy proposal & email content calendar

### Research (official deliverable)

| Area | Action |
| --- | --- |
| Onboard notes | Transcribe while fresh; future-you must understand them. |
| Prior proposal | Re-use what the client loved. |
| SMART goal | From the call or create one (ChatGPT + Google OK). |
| Audience | Build a persona from the client’s **favourite customer**. |
| Industry email norms | Pillars, formats, legal limits they might have missed. |
| Brand kit | **Canva** — logo, colours, fonts (see steps + screenshots below). |

### Canva brand kit — steps

| Step | Instruction |
| --- | --- |
| 1 | Open Canva → **Brand** (left). |
| 2 | Existing PID with CI: pick the right kit (**53JK** has multiple — match carefully; **Harvard** kits need care). New PID: **Add new**. |
| 3 | Name kit with **PID**, upload **PNG** logos from partner, add palette + fonts. |
| 4 | No brand guide? Pull **hex** + **font** from the website (**Inspect** / F12). |

## Content calendar creation

| Step | Action |
| --- | --- |
| 1 | Open **[template]** → **make a copy**. |
| 2 | Name: `[Business Name] [Month–Month] Email Content Calendar`. |
| 3 | Start with a **Welcome Email Series** (e.g. up to **6** emails in a drip). |
| 4 | Use **ChatGPT** prompts — paste **content strategy** + research context. |
| 5 | Validate ideas are possible in-platform (many **automations** are **not** supported). |
| 6 | Add ideas to the calendar. |
| 7 | Send **Anna / Anju / Riya** the link for QA; incorporate doc comments. |
| 8 | Download calendar as **PDF**. |

## Drafting start-up content

We begin email creation on the **1st of the month** following the order, while still covering **month zero** with **welcome** emails.

| Step | Action |
| --- | --- |
| Volume | **2 emails / month** (standard product). |
| Build | Open business in **Campaigns Pro** → create email. |
| Assets | Create **CIs** as needed. |
| Close | **Close out tasks** when done. |
