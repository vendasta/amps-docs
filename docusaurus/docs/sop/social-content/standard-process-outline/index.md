# Standard Process Outline

## Bringing In New Accounts

### Round Robin — team (5 roles)

| Role | Where it’s assigned |
| --- | --- |
| **Social drafter** | **RR** tab in the **social** lifeline |
| **Social QA** | Same as drafter |
| **Custom image creator** | **CI** tab in the lifeline |
| **Blog drafter** | **SMM Blog Projects** tab in the **longform** lifeline |
| **Blog QA** | **SMM Blog Projects** tab in the longform lifeline |

*One Standard round-robin uses all five roles above.*

### Due dates (Standard)

| Workstream | Due |
| --- | --- |
| **Social — drafting** | **1 business day** after assignment |
| **CI (custom images)** | **2 days** after assignment |
| **Blog — drafting** | **5 days** after assignment |

## Custom Images — scope & rules

| Topic | Guidance |
| --- | --- |
| **Deliverables (typical)** | **2–3 CI**, **2 unique stories**, **1–2** story repost of CI, **blog banner** |
| **Engagement** | CI + IG stories are **engagement-only** where applicable — **never a Promo** unless an exception is stated |

---

## Unique vs. IG stories

| Area | Guidance |
| --- | --- |
| **Deliverables** | Same ballpark as CI: **2–3 CI**, **2 unique stories**, **1–2** reposts, **banner** |
| **Your job** | In the folder, gather what the **CI creator** needs to build the image |
| **Tooling** | More ideas in **Canva** / **Adobe Express** |

### Topic ideas (not exhaustive)

| Idea |
| --- |
| How it’s made |
| Fun fact |
| Did you know |
| Reviews (different from FB post) |
| Blog highlight |
| Show updates, hours, etc. |
| Product highlight |
| Quotes |
| Tips |

## Custom Images — Canva & files

| Step | Action |
| --- | --- |
| 1 | In **PID/Businesses**, create a folder for the **current month/year** |
| 2 | Create **three** design files: **Month– CI**, **Month– IG Story**, **Month– Blog Banner** |
| 3 | Link the month folder in the **CI LL** → **Canva Link** column |
| 4 | After proofing, **download** each file and upload to the right **Google Drive** folders in the drafting doc → place in **calendar** |
| 5 | **Blog thumbnail** goes to the **blog image** drive |

## Blogs (quarterly)

| Topic | Detail |
| --- | --- |
| **Cadence** | **1 quarterly** blog, usually with the **first** calendar |
| **Topics** | Decide how drafts should **look** with the content approach you’re using |

### Status workflow (Blogs LL)

| Status | Meaning |
| --- | --- |
| **In progress** | Blog is being implemented — update **Blogs LL** after drafting |
| **Ready to finish** | Draft done; **CI/ID** images in; info, description, and **SEO** in **CRM** — e.g. coordinated in the project with social |
| **Complete** | Sent for **approval** with the rest of the calendar |

| Item | Detail |

| **Folders** | Each account gets a folder for the **quarter** they’re working in |
| **Breadcrumb** | `SELF-INNER SERVICES > Blogs >` *(use the path your UI shows)* |
